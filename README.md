# 📊 E-Commerce Return Rate Reduction Analysis

## 📌 Project Description

This project is an end-to-end Data Analytics project that analyzes an e-commerce returns dataset using Python, MySQL, and Power BI. The objective is to understand customer purchasing behavior, identify return patterns, evaluate business profitability, and analyze sustainability metrics to support data-driven business decisions.


# 🎯 Project Objectives

- Analyze e-commerce orders and returns
- Understand customer purchasing behavior
- Identify major return reasons
- Analyze revenue and profitability
- Evaluate customer demographics
- Measure sustainability metrics
- Create an interactive Power BI dashboard
- Generate business insights for decision-making
